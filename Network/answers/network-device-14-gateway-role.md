## Q. 게이트웨이의 역할과 라우팅 테이블에서의 확인 방법을 설명하시오.

### 핵심 답변

**게이트웨이(gateway)**는 서로 다른 통신망, 프로토콜을 사용하는 네트워크 간의 통신을 가능하게 하는 **관문 역할**을 하는 컴퓨터나 소프트웨어를 두루 일컫는 용어입니다.

사용자는 인터넷에 접속하기 위해 수많은 톨게이트인 게이트웨이를 거쳐야 하며, 게이트웨이는 서로 다른 네트워크상의 통신 프로토콜을 변환해주는 역할도 합니다.

### 보충 설명

**라우팅 테이블에서 게이트웨이 확인:**
- Windows 명령 프롬프트에서 `netstat -r` 명령어 실행
- IPv4 경로 테이블에서 게이트웨이, 인터페이스 항목 확인 가능

**게이트웨이 예시:**
```
A 네트워크 → [게이트웨이] → B 네트워크 → [게이트웨이] → C 네트워크
```
서로 다른 프로토콜을 사용하는 네트워크 간의 통신을 중재

### 키워드
- 게이트웨이, 라우팅 테이블, 프로토콜 변환, netstat -r
