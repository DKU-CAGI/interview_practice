## Q. 허브, L2 스위치, 라우터의 차이점을 계층 기준으로 설명하시오.

### 핵심 답변

| 기기 | 처리 계층 | 참조 주소 | 참조 PDU | 특징 |
|------|---------|---------|---------|------|
| NIC, 리피터, AP | 물리 계층 | - | 비트 | AP는 패킷을 복사 |
| L2 스위치, 브리지 | 데이터 링크 계층 | MAC 주소 | 이더넷 프레임 | MAC 주소 테이블로 스위칭 |
| 라우터, L3 스위치 | 인터넷(네트워크) 계층 | IP 주소 | IP 패킷 | 라우팅 테이블로 경로 결정 |
| L7 스위치 | 애플리케이션 계층 | URL, HTTP 헤더, 쿠키 | 메시지 | 로드밸런서 역할 |

### 보충 설명

**상위 계층을 처리하는 기기는 하위 계층도 처리 가능하지만, 그 반대는 불가능합니다.**
- L7 스위치는 애플리케이션 계층부터 물리 계층까지 모두 처리 가능
- AP(Access Point)는 물리 계층밖에 처리하지 못함

**리피터**: 들어오는 약해진 신호를 증폭하여 전달하는 장치. 광케이블 보급으로 현재는 잘 사용하지 않음
