## Q. OSI 7계층과 TCP/IP 4계층의 차이점을 설명하시오.

### 핵심 답변

| TCP/IP 4계층 | OSI 7계층 |
|-------------|----------|
| 애플리케이션 계층 | 애플리케이션 계층 |
| ↑ (3개를 하나로) | 프레젠테이션 계층 |
|  | 세션 계층 |
| 전송 계층 | 전송 계층 |
| 인터넷 계층 | **네트워크 계층** (명칭 다름) |
| 링크 계층 | 데이터 링크 계층 |
| ↑ (2개를 하나로) | 물리 계층 |

**차이점:**
1. OSI는 **애플리케이션 계층을 3개(애플리케이션, 프레젠테이션, 세션)로 분리**
2. OSI는 **링크 계층을 2개(데이터 링크, 물리)로 분리**
3. OSI는 인터넷 계층을 **네트워크 계층**이라고 부름

### 보충 설명

면접 답변 예시:
> "TCP/IP 계층과 달리 OSI 계층은 애플리케이션 계층을 세 개로 쪼개고 링크 계층을 데이터 링크 계층, 물리 계층으로 나눠서 표현하는 것이 다르며, 인터넷 계층을 네트워크 계층으로 부른다는 점이 다릅니다."
