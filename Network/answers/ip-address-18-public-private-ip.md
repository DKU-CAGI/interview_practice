## Q. DHCP의 IP 주소 할당 과정(DORA)을 설명하시오.

### 핵심 답변

**DHCP(Dynamic Host Configuration Protocol)**는 IP 주소 및 기타 통신 매개변수를 자동으로 할당하기 위한 네트워크 관리 프로토콜입니다.

이 기술을 통해 네트워크 장치의 IP 주소를 수동으로 설정할 필요 없이, 인터넷에 접속할 때마다 자동으로 IP 주소를 할당받을 수 있습니다.

**DORA 과정:**

| 단계 | 이름 | 방향 | 설명 |
|------|------|------|------|
| 1 | **Discover** | 클라이언트 → 브로드캐스트 | DHCP 서버를 찾음 |
| 2 | **Offer** | DHCP 서버 → 클라이언트 | IP 주소 제안 |
| 3 | **Request** | 클라이언트 → DHCP 서버 | 제안된 IP 주소 요청 |
| 4 | **Acknowledge** | DHCP 서버 → 클라이언트 | IP 주소 할당 확인 |

### 보충 설명

- 많은 라우터와 게이트웨이 장비에 DHCP 기능이 있으며, 이를 통해 **대부분의 가정용 네트워크에서 IP 주소를 할당**
- DHCP를 사용하면 IP 충돌 없이 자동으로 관리됨
- 임대 방식으로 IP 주소를 사용하므로 일정 시간 이후 갱신 필요
