## Q. 처리량(throughput)과 대역폭(bandwidth)의 차이를 설명하시오.

### 핵심 답변

| 구분 | 처리량(Throughput) | 대역폭(Bandwidth) |
|------|-------------------|------------------|
| 정의 | 링크 내에서 **성공적으로 전달된** 데이터의 양 | 주어진 시간 동안 네트워크 연결을 통해 **흐를 수 있는 최대** 비트 수 |
| 관계 | 실제 성능 (처리량 ≤ 대역폭) | 이론적 상한선 |
| 단위 | bps(bits per second) | bps(bits per second) |

- **트래픽이 많아졌다** = 링크 내에 흐르는 데이터가 많아졌다
- **처리량이 많아졌다** = 처리되는 트래픽이 많아졌다

### 보충 설명

처리량에 영향을 주는 요소:
- 사용자가 많이 접속할 때마다 커지는 **트래픽**
- 네트워크 장치 간의 **대역폭**
- 네트워크 중간에 발생하는 **에러**
- 장치의 **하드웨어 스펙**
