## Q. 네트워크란 무엇인가? 노드(node)와 링크(link)의 개념을 설명하시오.

### 핵심 답변

네트워크란 **노드(node)와 링크(link)가 서로 연결되어 리소스를 공유하는 집합**을 의미합니다.

- **노드**: 서버, 라우터, 스위치 등 네트워크 장치
- **링크**: 노드 사이를 연결하는 유선(이더넷 케이블 등) 또는 무선(Wi-Fi 등) 연결

### 보충 설명

좋은 네트워크의 조건:
- **높은 처리량(throughput)**: 많은 트래픽을 처리할 수 있음
- **짧은 지연 시간(latency)**: 요청이 빠르게 처리됨
- **낮은 장애 빈도**: 안정적으로 운영됨
- **좋은 보안**: 외부 공격에 대비

### 키워드
- 노드, 링크, 처리량, 지연 시간
