## Q. NAT란 무엇이고, 공인 IP와 사설 IP를 어떻게 처리하는지 설명하시오.

### 핵심 답변

**NAT(Network Address Translation)**는 패킷이 라우팅 장치를 통해 전송되는 동안 **패킷의 IP 주소 정보를 수정하여 IP 주소를 다른 주소로 매핑**하는 방법입니다.

IPv4 주소 체계만으로는 많은 주소들을 모두 감당하지 못하는 단점이 있는데, 이를 해결하기 위해 NAT로 **공인 IP와 사설 IP로 나눠서** 많은 주소를 처리합니다.

| 구분 | 공인 IP | 사설 IP |
|------|--------|--------|
| 범위 | 인터넷 전체에서 유일 | 내부 네트워크에서만 사용 |
| 예시 | 121.165.151.200 | 192.168.0.x |
| 접근 | 외부에서 직접 접근 가능 | 외부에서 직접 접근 불가 |

### 보충 설명

**NAT 동작 방식:**
```
사설 IP (192.168.0.2~5) → [NAT 장치/라우터] → 공인 IP (121.165.151.200) → 인터넷
```
여러 대의 사설 IP를 가진 장치가 하나의 공인 IP로 인터넷에 접속 가능

**NAT의 장점:**
- 공유기와 NAT 기능으로 여러 PC를 하나의 공인 IP로 인터넷 연결 가능
- 내부 네트워크의 IP 주소가 외부에 드러나지 않아 **보안** 효과

**NAT의 단점:**
- 여러 명이 동시에 접속하므로 실제로 접속하는 호스트 숫자에 따라 **접속 속도가 느려질 수** 있음

NAT를 가능하게 하는 소프트웨어: ICS, RRAS, Netfilter 등
