# CS 인터뷰 스터디

개발자 면접 대비를 위한 CS 지식 정리 레포지토리입니다.

## 목차

| 주제 | 설명 |
|------|------|
| [Design Pattern](./Design-pattern/) | 디자인 패턴 |
| [Network](./Network/) | 디자인 패턴 |

## 진행 방식

- 매주 주제를 선정하여 질문과 답변을 정리합니다.
- 각 주제 폴더의 `Question.md`에 질문 목록을 작성합니다.
- 질문별로 개별 파일을 생성하여 상세 답변을 정리합니다.

## 파일 구조

```
cs-interview/
├── README.md
├── Design-pattern/
│   ├── Question.md        # 질문 목록
│   └── answers/
│       └── singleton.md   # 질문별 상세 답변
└── {주제}/
    ├── Question.md
    └── answers/
```

## 답변 작성 규칙

- 핵심 개념을 간결하게 정리합니다.
- 코드 예시가 있으면 함께 첨부합니다.
- 꼬리 질문(Follow-up)을 함께 준비합니다.
